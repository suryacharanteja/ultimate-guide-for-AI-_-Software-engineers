# Internet Simulator (React)

> **Note:** This simulator models the TCP/IP stack augmented with the Physical Layer (L1), giving a complete view of the end-to-end data flow — from raw bit transmission on the wire all the way up to the Application Layer.

## Requirements

- Node.js 18+

## Getting Started

```bash
npm install
npm run dev
```

Open your browser at `http://localhost:5173`

## Other Commands

| Command | Description |
|---------|-------------|
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
